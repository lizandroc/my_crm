MyConnect Hub CRM — Setup & Deployment Guide

A multi-platform contact CRM with duplicate merging, interest matching and an AI discovery engine. Built with Hono + Cloudflare Pages + D1.

1. Run locally

npm install
npx wrangler d1 migrations apply webapp-production --local
npm run build
npx wrangler pages dev dist --d1=webapp-production --local --port 3000

Open http://localhost:3000 — use the Demo button to explore with sample data.

2. Get your Cloudflare API token

Create a token at dash.cloudflare.com/profile/api-tokens → Create Token → Custom token with these permissions:

  • Account → Cloudflare Pages → Edit
  • Account → D1 → Edit
  • Account → Account Settings → Read
Security: never commit your token to git or paste it into files. Provide it only as an environment variable at deploy time (below), or in your deployment platform's secret manager.

3. Deploy to Cloudflare Pages

# Set your token for this terminal session only (replace placeholder):
export CLOUDFLARE_API_TOKEN="PASTE_YOUR_TOKEN_HERE"

# One-time: create the Pages project + production database
npx wrangler pages project create myconnect-hub --production-branch main
npx wrangler d1 create webapp-production
#   → copy the database_id it prints into wrangler.jsonc

# Apply schema to production and deploy
npx wrangler d1 migrations apply webapp-production
npm run build
npx wrangler pages deploy dist --project-name myconnect-hub

Your app will be live at https://myconnect-hub.pages.dev.
